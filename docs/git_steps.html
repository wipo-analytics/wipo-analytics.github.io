Process - STEPS

  1. In Git make sure you are in master brach
  2. PULL to get latest version of master
  3. Create a new branch for the work you are about to do. Give the branch a name jazz_api, jazz_sql
  4. Make sure you are in your branch
  5. Make changes
  6. Commit and push on your branch
  7. Make pull request
  8. Merge to Master if you are sure it will not break anything
  9. Otherwise request review
